# Media Stream Analyzer & Transcoder (Web)

A modern web application for analyzing online media streams and preparing audio/video formats for download.

This project focuses on **frontend + backend engineering**, stream analysis, format selection, and media processing workflows.

---

## Overview

The application allows users to:
- Paste a media URL
- Analyze available audio and video formats
- Select audio or video quality
- Prepare downloads through a structured pipeline

Due to platform and legal constraints, direct downloading of copyrighted content is intentionally restricted in the web interface.

---

## Tech Stack

### Frontend
- React (Vite)
- Modern component-based UI
- Dark / Light theme support

### Backend
- Node.js
- Express
- Stream analysis utilities
- FFmpeg-based processing (local use)

---

## Features

- Media URL analysis
- Audio / Video format discovery
- Audio vs Video selection tabs
- Quality selection UI
- Premium dark & light mode UI
- Clean, scalable architecture
