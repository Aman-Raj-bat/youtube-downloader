# Media Stream Analyzer & Transcoder (Backend)

## Overview
A backend system built with **Node.js**, **Express**, and **FFmpeg** for analyzing media streams and performing audio/video processing.

This project focuses on backend engineering concepts such as stream handling, format analysis, and media transformation.

---

## Features
- Media stream analysis and metadata extraction
- Audio and video format discovery
- Direct stream downloads
- FFmpeg-based audio conversion (MP3)
- High-quality video + audio merging
- Clean, scalable backend architecture

---

## Tech Stack
- Node.js
- Express.js
- FFmpeg
- @distube/ytdl-core
